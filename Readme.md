# Tech Blog

### A test program for Beego framework

Install [Go][79bc4822] and [Beego][61d4224d]

  [79bc4822]: https://golang.org/ "Go to Go"
  [61d4224d]: http://beego.me/ "Go to Beego"

Clone this repo and run:

```
> bee run techblog
```

Open  ``http://localhost:8080`` on browser

Enjoy yourself!
